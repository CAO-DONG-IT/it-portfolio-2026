# IT学習ポートフォリオ（2026）

## 目的
未経験からIT職（ITサポート／ヘルプデスク／テスター／運用保守／初級エンジニア）への転職を目指し、学習成果を「見える化」して提示できる状態にする。

## 学習範囲
- Linux基礎（操作・ログ確認・簡易トラブルシュート）
- Java基礎（小規模コンソールアプリを継続改善）
- AWS基礎（第2週以降：EC2などの基本構築・運用の流れ）

## 成果物（アウトプット）
- Linux：
  - トラブルシュート手順書（Playbook）
  - コマンド集（用途／例）
  - 演習ログ、学習メモ
- Java：
  - コンソールアプリ（要件→実装→改善の履歴）
  - バージョンごとの変更点（リリースノート）
- AWS：
  - EC2構築手順（手順書）
  - セキュリティ／コスト注意点チェックリスト
  - 簡易構成メモ（図は後日追加可）

## 学習記録
日次ログは `weekly-log/` に記録する。

## 注意
個人情報（住所・電話番号・身分証等）は記載しない。
