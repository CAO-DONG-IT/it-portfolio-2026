## Console Todo（概要）
- v0.1：標準入力で 1 件のタスクを受け取り、空入力を判定して表示する最小構成を確認
- v0.2：メニュー操作（追加／一覧／終了）を実装し、タスク追加・一覧表示を可能にした
- v0.3：メニュー入力と削除番号の入力チェックを強化し、番号指定による削除機能を追加した

### 各バージョンの要点
- v0.1（2026-01-06）
  - 標準入力で 1 件のタスクを受け取る
  - 空入力（空文字）を判定して表示する

- v0.2（2026-01-07）
  - メニュー操作（1:追加 / 2:一覧 / 0:終了）
  - タスク追加（空入力は登録しない）
  - タスク一覧表示（番号付き）
  - メソッド分割による可読性改善（`showMenuAndRead` / `addTask` / `listTasks`）

- v0.3
  - 追加：タスク文字列を登録（空文字は拒否）
  - 一覧：タスクを番号付きで表示（空ならメッセージ表示）
  - 削除：番号指定で削除（数字以外・範囲外は拒否）
  - 終了：アプリ終了
  - メニュー入力（0/1/2/3）および削除番号の入力チェックを強化し、誤入力時はメッセージを表示して再入力できるようにした

### 補足
- 現在は新規改修を一時停止しており、既存の v0.1〜v0.3 を成果物として保持している
- 再開時は、入力処理や構成整理を段階的に見直す想定
