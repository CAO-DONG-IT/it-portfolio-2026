# Console Todo v0.1（2026-01-06）

## できること
- 標準入力で1件タスクを受け取り、空入力を判定して表示する

## 実行方法（WSL/Ubuntu）
```
javac Main.java
java Main
```

## 操作例

入力：

`学校に行く`

出力：

`登録しました: 学校に行く`

## 学び（メモ）

`trim()`：前後空白を除去

`isEmpty()`：空文字（長さ0）判定

## 注意：

全角スペース（`U+3000`）が混入すると `illegal character: '\u3000'` でコンパイルエラーになる
