# Console Todo v0.2（2026-01-07）

## できること
- メニュー操作（`1)追加 / 2)一覧 / 0)終了`）
- タスク追加（空入力は登録しない）
- タスク一覧表示（番号付き）
- メソッド分割による可読性改善（`showMenuAndRead / addTask / listTasks`）

## 実行方法（WSL/Ubuntu）
```
javac Main.java
java Main
```

## 操作例
### 追加

入力：
```
1
学校に行く
```
出力：
`登録しました: 学校に行く`

### 一覧

入力：
```
2
```

出力：
```
=== タスク一覧 ===

学校に行く`
```
### 終了

入力：
```
0
```

出力：
```
終了します。
```

## 学び（メモ）

`ArrayList<String>`：タスクを複数保持できる（可変長リスト）

`equals()`：文字列比較は == ではなく `equals` を使う

`trim()`：前後空白を除去して入力ゆれを減らす

`isEmpty()`：空文字（長さ0）判定

`return;`：メソッド（例：`addTask`）を途中で終了し、`main` のループに戻る

`print / println`：`print` は改行しないため、表示が連結して見えることがある

## 注意
```
全角スペース（U+3000）が混入すると illegal character: '\u3000' でコンパイルエラーになる

main と private static メソッドはクラス直下に置く（括弧の階層が崩れるとビルドエラーになりやすい）
```
