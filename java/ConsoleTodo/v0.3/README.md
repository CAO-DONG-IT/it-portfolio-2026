# Console Todo v0.3

## 概要
コンソール上で TODO（タスク）を管理する小さなアプリです。  
メニュー選択は **半角数字（0/1/2/3）** のみ受け付け、誤入力時はメッセージを出してやり直せます。
v0.3 の追加点：メニュー入力（0/1/2/3）と削除番号の入力チェックを強化し、誤入力時はメッセージを出して再入力できるようにしました。

## できること
- 追加：タスク文字列を登録（空文字は拒否）
- 一覧：タスクを番号付きで表示（空ならメッセージ表示）
- 削除：番号指定で削除（数字以外・範囲外は拒否）
- 終了：アプリ終了

## 実行方法
### IntelliJ IDEA
`Main.java` を実行

## 使い方
起動するとメニューが表示されます。

- `1`：追加  
  タスクを入力して登録します（空の入力は拒否）
- `2`：一覧  
  現在のタスクを表示します
- `3`：削除  
  一覧を表示した後、削除する番号を入力します  
  数字以外は「数字で入力してください。」  
  範囲外は「範囲外です。1〜N を入力してください。」
- `0`：終了

※ メニュー入力は **半角数字（0/1/2/3）** のみ有効です（全角の「１」や「a」などは拒否）
