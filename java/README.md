# ConsoleTodo（Java）
小規模なコンソールアプリを段階的に拡張し、実装・実行・証跡（再現性）をセットで残す。
## 目標
- CRUD（作成・参照・更新・削除）を段階的に実装する
- READMEだけで実行手順が再現できる状態にする
- 学習ログ（weekly-log）から成果物へ最短で辿れるようにする
  
## 実行方法
### IntelliJ IDEA
1. プロジェクトを開く
2. `Main.java` を開く
3. ▶（Run）で実行
4. コンソールの指示に従って入力

### WSL / Ubuntu
```bash
javac Main.java
java Main
```

## バージョン記録
- v0.1：起動確認（標準入力 → 空判定 → 出力）
- v0.2：メニュー＋追加／一覧
- v0.3：更新／削除（入力チェック含む）
- v0.4（予定）：ファイル保存／読み込み
- v0.5（予定）：例外処理／リファクタリング

## 証拠ルール
-「動く状態」でコミットする
- 各バージョンのREADMEに「実行手順」と「操作例（入出力）」を残す
- スクリーンショットを残す場合は `screenshots/ `を使用する（存在する場合）
