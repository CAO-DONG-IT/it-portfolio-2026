# Java / Screenshots（証跡）

このディレクトリは、Java（ConsoleTodo）の動作確認結果を「証跡」として保存するための場所です。
面接時に「動いた事実」を短時間で提示できるよう、ファイル名と参照ルールを統一します。

## 何を保存するか
- 実行結果（追加／一覧／更新／削除など）のコンソール表示
- 入力チェック（不正入力・範囲外・全角混入など）の挙動
- 例外発生時のメッセージ（try-catch の確認など）
- 重要な変更の前後比較（改善が分かるもの）

## 命名ルール
以下の形式で命名し、検索しやすくします。

`YYYY-MM-DD_v0.x_<feature>_<action>.png`

例：
- `2026-01-06_v0.1_input_nonempty_ok.png`
- `2026-01-07_v0.2_menu_add_list_ok.png`
- `2026-01-07_v0.2_validation_fullwidth_ng.png`

## README からの参照
各バージョンの README（例：`java/ConsoleTodo/v0.1/README.md`）の「操作例」や「注意」から、
該当スクリーンショットへリンクします。

## 注意（公開リポジトリ運用）
- 個人情報（氏名、メール、画面通知など）が写り込まないようにする
- 画面の一部に不要な情報がある場合は、トリミングしてから保存する
