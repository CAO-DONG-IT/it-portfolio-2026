# Linux：dateコマンドとシンボリックリンクの基本 (2026-01-18)

## 1. dateコマンドの書式と相対時間
日付表示のカスタマイズと、過去/未来の時刻計算方法。

### カスタム書式 [フォーマット / Format / 格式]
- **基本：** `date +"%Y-%m-%d %H:%M:%S %Z"`
- **注意：** `+` とダブルクォートの間にスペースを入れないこと（エラーの原因）。

### 相対時間の表示
- **1日前：** `date -d "-1 day"`
- **1ヶ月前：** `date -d "-1 month"`

## 2. シンボリックリンク [シンボリックリンク / Symbolic Link / 符号链接]
ファイルやディレクトリへのショートカット作成。

### 成功した手順
- **コマンド：** `ln -s /etc/timezone ~/`
- **確認：** `ls -l` で `timezone -> /etc/timezone` と表示されれば正解。

### 重要なルール
作成時は、リンク先を**絶対パス [絶対パス / Absolute Path]** で指定するのが最も安全。相対パスで指定すると、リンクファイル自体の場所から見たパスとして解釈され、循環参照エラーが発生するリスクがある。
