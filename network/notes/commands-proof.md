# 基本コマンド再現ログ（2026-01-07）

## 疎通確認（IP直）
- コマンド：`ping 1.1.1.1`
- 結果：送信=4 受信=4 損失=0（0%）、時間=7〜8ms、TTL=56
- 一言：IP疎通はできた

## DNS確認
- コマンド：`nslookup example.com`
- 結果：DNS=8.8.8.8（dns.google）、example.com → 104.18.27.120 / 104.18.26.120（IPv6も返却）
- 一言：名前解決はできた

## 経路確認
- コマンド：`tracert 1.1.1.1`
- 結果：最終hopで 1.1.1.1 に到達（途中に * があるが最終到達）
- 一言：経路確認はできた（途中ルータの応答なしはあり得る）

## HTTP確認
- コマンド：`curl.exe -I https://example.com`
- 結果：HTTP/1.1 200 OK（Server: cloudflare）
- 一言：HTTP到達はできた（200 OK）

## まとめ（切り分け結論）
- IP疎通：OK / DNS：OK / 経路：OK / HTTP：OK（200）
- 次に確認するもの：IPv4/サブネット（CIDR）とローカルIP設定の理解
